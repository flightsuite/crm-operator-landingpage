## Google Search Console Setup Guide

### Step 1: Verify Domain Ownership
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click "Add Property"
3. Select "URL prefix" and enter `https://flightsuite.ai`
4. Choose verification method (recommended: HTML file upload or DNS record)

### Step 2: Submit Sitemap
1. In Search Console, go to "Sitemaps" in the left menu
2. Enter `sitemap.xml` in the "Add a new sitemap" field
3. Click "Submit"

### Step 3: Request Indexing
1. Go to "URL Inspection" tool
2. Enter the homepage URL
3. Click "Request Indexing"
4. Repeat for all 8 pages in sitemap

### Step 4: Monitor Performance
- Check "Performance" report weekly
- Monitor "Coverage" for indexing issues
- Set up email alerts for critical issues
