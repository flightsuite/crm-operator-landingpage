2

Get Your Location ID

Find your GoHighLevel Location ID:

  • Log in to your GoHighLevel account
  • Go to Settings > Business Profile
  • Copy your Location ID
3

Create PAT Token

Generate a Personal Access Token (PAT) in GoHighLevel:

  • Navigate to Settings > Integrations
  • Click on "API" or "Developer"
  • Select "Create New PAT Token"
  • Name it "FlightSuite"
  • Select required permissions (Contacts, Notes, Opportunities)
  • Copy the generated token immediately

âš ï¸ Save your PAT token securely. It won't be shown again.

4

Enter Credentials

In the FlightSuite extension:

  • Click the FlightSuite icon in Chrome
  • Paste your Location ID
  • Paste your PAT Token
  • Click "Connect" and start using FlightSuite!
