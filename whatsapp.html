Questions

Is SMS live now? What about WhatsApp? +
Yes — SMS text-to-CRM is live right now. Sign up above to start using it today. WhatsApp support is coming soon and will work the same way. Sign up now and you'll get WhatsApp access automatically when it launches.
Is the Chrome extension still free? +
Yes — always. The Chrome extension is free forever with unlimited use. Subscriptions only apply to SMS and WhatsApp text queries. You can keep using the extension without ever paying a cent.
What counts as one query? +
Every message you send to FlightSuite via SMS (and WhatsApp when it launches) counts as one query. That includes logging calls, updating contacts, and setting follow-ups. Replies and confirmations from FlightSuite don't count — only messages you send.
Which CRMs are supported? +
Currently HubSpot (69 tools) and GoHighLevel (39 tools) are fully supported. Salesforce integration is in development — you can join the waitlist to be notified when it launches.
Do I need to install anything? +
No. SMS works with your phone's built-in messaging app — nothing to install. Just pair your phone number and start texting FlightSuite. The Chrome extension is a separate product — also free, installed from the Chrome Web Store.
Can I cancel anytime? +
Yes. Cancel in seconds from your account portal. You keep your remaining queries until the end of the billing period. No fees, no questions asked.
