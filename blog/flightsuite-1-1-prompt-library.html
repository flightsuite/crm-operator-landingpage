How to Get the Update

If you already have FlightSuite installed, Chrome will automatically update the extension within 24-48 hours. To force an immediate update:

  • Open chrome://extensions in your browser
  • Enable Developer mode (top right toggle)
  • Click Update at the top left
  • Refresh any open CRM tabs

New to FlightSuite? Install it free from the Chrome Web Store and start using the prompt library immediately after connecting your CRM.
